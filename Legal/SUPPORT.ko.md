## ❓ 자주 묻는 질문

### 1. 클립보드가 자동으로 기록되지 않아요？

iOS 시스템 권한 제한 때문입니다. 다음을 확인해 주세요：
1. Quick Paste History 앱이 포그라운드에서 최소 한 번 실행되었는지 확인
2. 처음 실행 시 "붙여넣기 허용"을 탭했는지 확인 (시스템 권한 팝업)
3. 이전에 거부했다면 **시스템 설정 → Quick Paste History → 다른 App에서 붙여넣기**로 이동 → **허용**으로 변경
4. 클립보드 기록은 포그라운드에서 활성화된 경우에만 업데이트됩니다. 붙여넣을 때마다 앱 인터페이스 또는 키보드 확장 기능을 열어야 합니다.

---

### 2. 키보드 확장 기능 사용 방법

1. **시스템 설정 → 일반 → 키보드 → 키보드 → 새로운 키보드 추가**로 이동
2. **Quick Paste History** 선택
3. Quick Paste History 키보드 이름을 탭 → **전체 액세스 허용**
4. 텍스트 입력 화면에서 키보드 전환 키를 길게 누르고 위로 스와이프한 후 Quick Paste History 키보드로 전환하여 기록을 탐색하고 붙여넣기

---

### 3. iCloud 동기화가 안 돼요？

1. 모든 기기가 동일한 Apple ID로 로그인되어 있는지 확인
2. iCloud가 활성화되어 있는지 확인 (시스템 설정 → 프로필 → iCloud → 상태가 "로그인됨"으로 표시되면 OK)
3. 다른 기기에서 Quick Paste History를 열고 약 30초 기다리면 자동으로 동기화됩니다
4. 오랜 시간이 지나도 동기화되지 않으면 앱을 다시 시작해 보세요

> 동기화는 `NSUbiquitousKeyValueStore`를 기반으로 합니다. Apple은 수동 새로고침 인터페이스를 제공하지 않으며, 동기화는 보통 수십 초 내에 완료됩니다.

---

### 4. 모든 기록을 지우는 방법

앱 메인 페이지에서 오른쪽 상단의 설정 아이콘을 탭 → 설정 메뉴에서 "모든 기록 지우기"를 선택합니다. 이 작업은 취소할 수 없습니다.

---

### 5. Pro 기능 구매 및 사용 방법

1. 메인 화면 오른쪽 상단의 설정 아이콘을 탭 → **Pro 업그레이드**
2. Pro 버전을 선택하고 인앱 구매를 완료
3. 구매 후 기능이 자동으로 잠금 해제됩니다
4. 동일한 Apple ID의 기기에서 구매를 복원할 수 있습니다 (구매 페이지에서 "구매 복원" 탭)

---

### 6. Quick Fill(퀵 필)이란 무엇인가요？

Quick Paste History에서 자주 사용하는 정보를 미리 Quick Fill에 추가할 수 있습니다. 키보드 확장 기능에서 빠른 채우기 진입점을 제공하여 자주 사용하는 콘텐츠를 빠르게 붙여넣을 수 있습니다.

---

### 7. 검색 기능 사용 방법

메인 페이지 상단에 검색창이 있습니다. 키워드를 입력하여 기록의 텍스트 콘텐츠를 검색할 수 있습니다. 결과는 관련성 순으로 정렬됩니다.

---

### 8. 데이터는 안전한가요？

- 모든 클립보드 데이터는 기기 로컬과 iCloud 개인 공간에만 저장됩니다
- 개발자는 클립보드 내용에 접근할 수 없습니다
- 사용자 데이터를 수신하는 백엔드 서버는 없습니다
- 자세한 내용은 개인정보 처리방침을 참조해 주세요
